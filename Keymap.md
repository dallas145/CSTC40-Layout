## TapDance
1. TapDance(1)
    | Action | Result |
    |:---:|:---:|
    | On Tap | Ctrl + LEFT |
    | On Hold | Ctrl + Alt + LEFT |

2. TapDance(2)
    | Action | Result |
    |:---:|:---:|
    | On Tap | Ctrl + DOWN |
    | On Hold | Ctrl + Alt + DOWN |

3. TapDance(3)
    | Action | Result |
    |:---:|:---:|
    | On Tap | Ctrl + UP |
    | On Hold | Ctrl + Alt + UP |

4. TapDance(4)
    | Action | Result |
    |:---:|:---:|
    | On Tap | Ctrl + RIGHT |
    | On Hold | Ctrl + Alt + RIGHT |

5. TapDance(5)
    | Action | Result |
    |:---:|:---:|
    | On Hold | MO(1) |
    | On Double Tap | One Shot Modify Shift |
    | On Tap and Hold | MO(4) |
